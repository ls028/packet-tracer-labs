# Cisco Packet Tracer Networking Labs

This repository contains hands-on networking labs built in Cisco Packet Tracer to develop practical skills in routing, switching, and network configuration.

Each lab focuses on real-world concepts used in home, enterprise, and production networks.

---

## 🧠 Skills Demonstrated

* Static Routing
* DHCP Configuration
* Multi-Network Design
* CLI Configuration (Cisco IOS)
* Network Troubleshooting
* Connectivity Testing (Ping, Traceroute)

---

## 📁 Labs

### 🔹 Lab 3: Static Routing Between Two Routers

Configured two routers to communicate across three separate networks using static routes. Troubleshot connectivity issues and verified routing tables.

**Key Concepts:**

* Static routes (`ip route`)
* Multi-router topology
* Routing tables (`show ip route`)
* Troubleshooting with `show` commands

📂 Folder: `Lab-3-Static-Routing`

---

### 🔹 Lab 4: DHCP Server on a Router

Configured a router to act as a DHCP server for two LANs, automatically assigning IP addresses to all connected devices.

**Key Concepts:**

* DHCP pools (`ip dhcp pool`)
* Default gateway assignment
* DNS configuration
* Automatic IP addressing

📂 Folder: `Lab-4-DHCP`
